Phishing Email Detection & Awareness System

This project analyzes phishing email samples, identifies phishing indicators, and provides user awareness guidelines.

 Tools Used
- Parrot OS
- GNU nano
- Firefox
- whois

 Analysis Approach
1. Collected phishing email sample (.eml)
2. Opened and inspected the email
3. Identified phishing indicators (sender, links, attachments)
4. Investigated suspicious domain with whois
5. Documented findings and created awareness report

 Files
- reports/Phishing Email Detection & Awareness System — final report
- evidence/ — screenshots of email, link, WHOIS
- samples/ — phishing email .eml
